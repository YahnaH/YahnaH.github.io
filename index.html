💯 Visual FoxPro is the best programming language I learned so far 💯

This is a programming language that I have learned outside of class, but so far I had the best experience with it. It was very easy to learn, albeit having some learning curve along the way, it is beginner-friendly. Although the language was very easy to learn, this does not hinder it from creating useful data-centric programs. The forms and GUIs are a bit rudimentary as compared to some of its advanced successors like Visual Basic (also developed by Microsoft), but it competes with VB in a sense that it is simpler to use and can generate advanced programs despite it being "outdated". My experience with VB was rather challenging because of its complicated syntax and you have to link it to a third-party applications like Microsoft Access to store data and generate Database Tables.

If I were to pick a language, however, that was taught in class to be my favourite, I'd say it's R. I found R very useful in numerical computations and creating programs that would help solve complicated Mathematical equations such as Matrix manipulation and Series computation. It can require a lot of analysis and creativity along the way to think of a way to integrate and optimize the needed steps to solve equations in our lab exercises, but once your program works, it is the kind of euphoria I wouldn't trade for anything else.
